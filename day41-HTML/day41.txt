1.Web
    1.介绍
        Web就是网页，基于B/S的应用程序
        B: Browser 浏览器
        S: Server 服务器
    2.组成
        1.浏览器  代替用户向服务器发起请求，解析服务器返回的数据，图形化呈现给用户
        2.服务器  提供数据支持，处理网络请求
        3.网络通信协议  http/https ---> 超文本传输协议，规范数据在网络中的传输方式
    3.Web服务器
        1.产品
            Apache
            Tomcat
            IIS   --->   Internet Information Service
            Nginx(python)
            ...
        2.开发语言
            Java
            C
            PHP
            Python(Django, Flask...)
    4.Web浏览器
        1.产品(主流)
            Chrome          Google
            IE Edge         Microsoft
            Firefox         Mozilla
            Opera           Apple
            Safari          Opera
            划分依据：浏览器的内核（引擎）
                1.渲染引擎：解析HTML,CSS,渲染页面
                2.JS引擎：解析页面中脚本代码
                例：webkit Trident gecko blink
        2.前端开发使用的技术
            1.HTML (通过标记书写页面的结构)
            2.CSS  (修饰和美化页面，实现页面布局)
            3.JavaScript (实现页面动效，实现交互)
            4.Angular Vue Node.js React 前端框架

2.HTML 概述
    1.HyperText Markup Language
        超文本标记语言
        作用：通过标记的形式书写页面结构并且填充内容
    2.标记
        也称为标签、元素，标记页面结构和内容，配合CSS实现页面布局和美化
        标记以<>为标志
    3.网页文件在计算机中以.html或.htm为后缀
        网页运行工具：浏览器
        网页开发工具：VSCode Editplus Sublime Hbuilder...
        网页调试工具：浏览器自带开发者工具，用于调试前端代码，可以在浏览器中右键“检查”打开，或键盘F12

3.HTML 基础语法
    1.标签（标记、元素）
        1.标签都使用<>表示，区分普通文本
        2.标签的分类
            1.双标签
                有开始有结束，成对出现
                语法：
                    <开始标签>
                        标签内容（文本或者嵌套标签）
                    </结束标签>
            2.单标签
                只有开始标签，没有结束标签
                语法：
                    <标签名>
                    或
                    <标签名/>
                例：
                    <br>或<br/>  表示换行
        3.标签嵌套 
            在双标签中使用其他的标签作为内容
        4.标签属性
            主要用来修饰当前标签，或者补充标签信息
            语法：
                <开始标签 属性名="值" 属性名="值">
            注意：
                1.属性必须写在开始标签中
                2.属性值使用引号表示
                3.多组属性之间使用空格间隔
        5.HTML 语法规范
            1.标签名不区分大小写，建议使用小写
            2.标签名书写错误或双标签缺少闭合时，浏览器都不报错。书写错误会以新标签形式展示，丧尸原有标签的功能，缺少闭合，浏览器会自动添加闭合标签，可能影响页面结构
            3.HTML中的注释语法
                <! -- 注释内容 -->
                注：注释不能嵌套使用           

4.HTML 标签介绍
    1.完整网页结构
        1.<!doctype html>
            HTML5提供的文档类型声明方式，以便浏览器能够正常解析HTML标签并且采用标准方式渲染页面
        2.<html></html>
            表示文档的开始和结束
        3.<head></head>
            表示网页头部，可以定义网页的标题，字符集，选项卡小图标，网页开发信息，关键字，外部资源的引入
        4.<body></body>
            表示网页主体，窗口区域，有关网页主要内容的展示都在body中书写
    2.网页头部设置<head></head>
        1.字符编码
            <meta charset="utf-8">
        2.网页标题
            <title>百度一下，你就知道</title>
        3.选项卡图标
            <link rel="shortcut icon" href="图片路径" type="image/x-icon">
            标签属性：
                rel：设置外部资源的类型
                href：指定资源路径
                type：文件的类型和格式
    3.网页主体内容<body></body>
        1.文本标签
            1.标题标签
                <hn></hn>
                n取1~6  代表h1~h6 六级标题
                标题自带文本加粗效果，字号从1~6逐级减小
            2.段落标签
                <p></p>
                以段落形式显示文本内容
            3.普通的文本标签
                1.行分区
                    <span></span>
                    可以用于对特殊文本的样式处理
                2.<label></label>
                    单独使用表示普通文本，可以结合表单控件，实现文本与控件的绑定
                3.文本格式标签
                    加粗：<b></b>   <strong></strong>
                    斜体：<i></i>   
                    删除线：<s></s>
                    下划线：<u></u>
                    上标：<sup></sup>
                    下标：<sub></sub>
        2.换行与空格
            1.浏览器会忽略代码中的换行和多余的空格，同一解析为一个空白字符
            2.换行标签：<br>
            3.字符实体：基于HTML标签语法，浏览器解析时，按照<>解析标签与内容，导致特殊符号无法正常显示，需要借助字符实体，实现特殊字符的正常显示
                1.< : &lt; ---> less than
                2.> : &gt; ---> greater than
                3.空格 : &nbsp;
                4.版权符号© : &copy;
                5.人民币符号￥ : &yen;
        3.水平线和块分区标签
            1.水平线 <hr>
            2.块分区标签 (重点)
                <div></div> 容器标签，用于页面的结构划分和模块划分
        4.元素的层次结构
            标签可以嵌套使用，出现嵌套时，标签之间的关系分为以下几种
                1.外层元素称为父元素，内层元素称为子元素
                2.多层嵌套时，外层所有的标签都可以称为内层元素的祖先元素，内层元素统称为后代元素
                  后代元素又可以分为直接子元素和间接子元素
                3.同级关系称为兄弟元素
        5.列表标签
            1.列表，采用有序或无序的方式从上到下分条展示数据
            2.分类
                1.有序列表(ordered list)
                    语法：
                        <ol>
                            <li>list item 列表项</li>
                        </ol>
                    默认以阿拉伯数字作为项目符号
                2.无序列表(unordered list)
                    语法：
                        <ul>
                            <li></li>
                        </ul>
                    默认以实心圆点作为项目符号
            3.列表的标签属性
                1.有序列表 - ol
                    1.type
                        设置项目符号，默认为阿拉伯数字
                        取值：
                            1
                            a
                            A 
                            i 
                            I 
                    2.start
                        设置项目符号从第几个字符开始标识
                        取值： 
                            无单位的数字，表示从第几个字符显示
                2.无序列表 - ul
                    type:设置项目符号类型
                        取值：  
                            disc(默认值) 实心圆点
                            circle 空心圆
                            square 实心方块
                            none 取消项目符号
            4.列表的嵌套
                在列表中使用另外的列表
                语法：  
                    <ul>
                        <li>
                            <ul>
                                <li></li>
                            </ul>
                        </li>
                    </ul>
