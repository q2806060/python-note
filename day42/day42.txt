1.图像与超链接
    1.URL（路径）
        1.Uniform Reasource Locator
            统一资源定位符，表示资源文件所在位置
        2.组成
            协议 域名 文件目录 文件名
            例：
                http://www.baidu.com
                http://www.baidu.com/img/pet/001.jpg
        3.分类
            1.绝对路径
                从根目录开始逐级查找文件，所形成的路径就是绝对路径
                以盘符为标志，或以/为开头
                注：本地路径，协议使用file:///表示，可以省略
                2.网络路径必须添加协议http://
                3.区分正反斜杠，'/'表示路径，'\'是转义字符的标志
            2.相对路径
                从当前文件所在的目录文件夹开始查找
                使用../表示回到上一级目录
                ./表示当前目录文件夹
        4.注意
            1.网络URL中不能出现中文
            2.URL严格区分大小写

    2.图像 
        在网页中插入一张图片
        语法： <img src="URL">
        属性：
            1.src:必填，指定图片路径
            2.width:取像素值，设置图片显示宽度，默认按照原始尺寸显示
            3.height:设置图片高度
            4.title:设置图片标题，鼠标悬停在图片上时显示
            5.alt:设置图片加载失败时的提示文本

    3.超链接
        1.可以实现资源文件的跳转
        2.语法：    
            <a href="URL">超链接内容(文本，图片)</a>
        3.使用
            超链接分为两种状态，默认带有不同样式
            访问前：蓝色文本+下划线
            访问后：紫色文本+下划线
        4.属性
            1.href：必填，指定连接地址
                   href=""表示链接到本页，相当于刷新
                   href="#"表示链接至本页，不会刷新页面，但是会改变页面的URL，自动在URL末尾添加#
                   href="javascript:viod(0)"不会造成页面刷新，也不会改变URL
            2.target：设置目标文件的打开方式，默认在当前窗口中打开
                     取值：
                        1._self默认值，在当前窗口打开
                        2._blank新建窗口打开
        5.锚点链接
            1.作用：跳转到指定文件的指定位置
            2.语法：    
                1.定义锚点
                    <a name=""></a>
                    在文档指定位置添加空的超链接标签，设置name属性，定义锚点名称

2.表格标签
    1.表格采用结构化的行和单元格存储和显示数据
    2.语法：    
        1.表格标签<table></table>
        2.创建行 <tr></tr>
        3.行中嵌套单元格<td></td>
    3.标签属性
        1.table 标签属性 
            1.border : 取像素值，设置表格边框
            2.bgcolor : 设置表格的背景颜色，取英文单词
            3.align : 设置表格的显示位置，默认居左，可取left/center/right
            4.width : 取像素值，设置表格宽度，默认表格尺寸有内容长短决定
            5.height : 取像素值，设定表格高度
            6.cellspacing : 设置边框与边框之间的距离
            7.cellpadding : 设置内容区域与边框之间的距离
        2.tr 标签属性
            1.bgcolor : 设置当前行的背景颜色
            2.align : 设置当前行中内容的水平对齐方式，默认居左，可取left/center/right
            3.valign : 设置内容的垂直对齐方式，默认垂直居中，可取top/middle/bottom
        3.td 标签属性 
            1.bgcolor : 设置单元格的背景颜色
            2.align : 设置单元格内容的水平对齐方式
            3.valign : 设置单元格内容的垂直对齐方式
            4.width/height : 设置单元格的宽高
    4.单元格合并（重点理解）
        1.单元格合并需要给td添加属性 ： colspan rowspan 
        2.属性介绍
            1.colspan : collum-span 跨列合并，横向合并当前行中的单元格，取无单位的整数，表示包含自身单元格在内，跨越几个单元格显示
            2.rowspan : 跨行合并，向下合并单元格，不影响当前行，会影响后面行中单元格的显示
        3.注意
            1.不管是跨行还是跨列，只要发生单元格合并，就需要调整表格结构，删除多余的单元格
            2.跨列合并，删除当前行中多余的单元格
            3.跨行合并，删除其后行中多余的单元格
    5.表格行分组
        1.<thead></thead>（表头行分组）
            将若干行作为表头，添加至thead中
        2.<tfoot></tfoot>（表尾行分组）
        3.<tbody></tbody>（表格主体内容）
            默认表格中所有的行都会被添加至tbody中显示
    6.th标签
        用法与td一致，表示单元格，常用于表头单元格，th自带文本居中和加粗效果

3.表单
    1.作用：用于收集用户信息发送给服务器
    2.组成
        1.form 元素，表示表单
        2.表单控件（输入框，单选、复选按钮）
        使用：
            表单控件只负责采集数据
            表单元素负责最终发送给服务器
    3.语法：
        <form>
            表单控件 
        </form>
    4.form 的标签属性
        1.action : 设置数据的提交地址
        2.method : 设置数据的提交方式，默认为get，可取post
            get/post 请求方式的区别
            1.get
                1.数据会以参数的形式拼接在url后面，明文传输
                 例：http://www.baidu.com?name=zhangsan&pwd=123456
                2.明文传输，安全性较低
                3.参数形式拼接，对数据的大小有限制，不超过2kb
            2.post
                1.数据会被打包传输，外界不可见
                2.隐式传输，安全性较高
                3.没有大小限制
        3.enctype : encode-type 缩写，指定数据发送给服务器时的编码类型
            取值：
                1.application/x-www-form-urlencoded
                    默认值，对应默认的数据提交方式get
                    表示将表单中的数据转换成一个字符串，拼接到url后面
                2.如果表单中涉及图片、文件或音频等二进制数据的提交，数据提交方式必须使用post，同时制定编码类型，enctype取multipart/form-data
    5.表单控件 
        1.提供一系列可以采集用户信息，与用户交互的可视化组件
        2.分类
            1.文本框与密码框
                语法：
                    <input type="text">   --->   文本框
                    <input type="password">   --->   密码框
                属性：
                    1.type : 必填，指定控件类型
                    2.name : 必填，指定控件名称
                    3.value : 指定控件的值（最终提交给服务器的值）
                    4.maxlength : 设置输入框最大字符数
                    5.placeholder : 设置输入框的提示文本
                    6.disable:禁用控件，该属性没有值，只能看，不能改，并且不能提交给服务器
                    7.readonly:只能读，不能改，可以提交给服务器
            2.单选钮和复选框
                语法：
                    <input type="radio">  --->  单选钮
                    <input type="checkbox>  --->  复选框
                属性：  
                    1.type
                    2.name : 必填，一组按钮的控件名称必须保持一致
                    3.value : 必填，表示控件的值，最终发送给服务器
                    4.checked : 可以设置按钮默认选中
                        完整写法 checked="checked"
                        可以省略属性值，只写属性名
