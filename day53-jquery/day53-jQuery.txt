jQuery 使用
1.介绍
    jQuery是JS的工具库，对原生JS的属性和方法进行封装，提供一套简化的方法
    版本：
        1.xx.xx 兼容IE 6,7,8
        2.xx.xx/3.xx.xx 不再兼容IE 6,7,8

2.功能和优势
    1.简化了DOM操作（节点获取，创建，删除，添加）
        简化了样式处理
    2.简化事件处理
    3.Ajex更加完善
    4.提供大量可用的插件，支持自定义插件

3.使用
    1.引入jQuery文件
        <script src=""></script>
        注意：只有先引入文件，才能使用jQuery的方法
    2.jQuery对象 -$ / jquery 表示
        借助于$()对原生JS进行封装，返回jquery对象
        区分对象类型：  
            原生对象调用原生方法
            jquery对象调用jquery的方法，方法不能混用
    3.$()
        作用：
            1.获取元素对象
            2.原生DOM对象与jquery对象的转换
        获取元素对象：
            $()接收字符串参数，传入选择器名称
            根据选择器匹配元素并返回jquery对象
            选择器：
                标签 id class 后代 子代...
            例：
                $("h1")
                $(".c1")
                $("#box")
                $("div p")
    4.操作节点内容
        1.html()
            类似于原生JS中的innerHTML属性，设置或读取标签内容，可识别HTML语法
        2.text()
            类似于innerText，不能识别标签语法
        3.val()
            类似于value属性，读取或设置表单控件的值
        4.原生JS与jquery代码可以共存   
            juqery主要对DOM对象，事件进行封装
            原生JS对象与jquery对象互换转换：
            1.JS -> jQ
                $(div)传入原生JS对象，进行封装，返回jquery对象
            2.JQ -> JS
                $("选择器")返回jquery数组，存放原生对象从JQ数组中取元素，取出即为原生对象
                1.$("#d1")[0];
                2.$("#d1").get(0);
