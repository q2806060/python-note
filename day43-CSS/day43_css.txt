1.CSS
    1.Cascading Style Sheet 层叠样式表，修饰和美化页面，实现页面整体的排版布局
    2.使用方式
        1.行内样式（内联样式）
            语法：
                将CSS样式生命通过标签属性style书写在标签内部
                <标签名 style="CSS样式声明">
                CSS样式声明:
                    主要由CSS属性和对应的属性值组成
                    属性名:属性值;
                常用样式声明
                    1.color:red;
                        设置元素文本颜色
                    2.background-color:green;
                        设置元素背景颜色
                    3.font-size:32px;
                        设置元素字体大小，取像素值
        2.文档内嵌
            语法：
                借助于<style></style>标签，将样式与结构分离
                在<style>标签中书写CSS代码
                <style type="text/scc">
                    CSS选择器{
                        属性：值
                        ...
                    }
                </style>
            CSS选择器：
                匹配文档中的一类元素，为其设置样式
                例：标签选择器
                    根据指定的标签名匹配文档中所有的该元素，统一设置样式
                    p{
                        color:red;
                    }
            原则：
                <style></style>标签可以书写在任意位置，书写任意次，一般遵循“样式优先”
        3.外链引入样式表文件
            语法：
                创建外部样式表文件，后缀采用.css，文件中使用CSS选择器定义样式声明
                在HTML中使用<link>引入外部资源文件
                <link rel="stylesheet" href="url" type="text/css">
    3.语法规范
        1.属性名和属性值全部使用小写字母
        2.所有标点符号全部使用英文标点
        3.选择器中的{}必须写完整
        4.属性名与属性值书写必须正确，写错浏览器会提示错误信息
        5.注释采用/**/
    4.样式表特征
        1.层叠性
            不同的CSS样式在同一个元素上共同起作用
        2.继承性
            指子元素可以继承使用父元素或祖先元素的样式
            例： 
                1.大部分的文本属性都可以被继承
                2.块元素默认宽度与父元素保持一致
        3.样式表的优先级
            元素发生样式冲突时最终选用哪一种样式，由样式表的优先级决定
            1.行内样式优先级最高
            2.文档内嵌和外链定义的样式，优先级一致，最终结果取决于代码书写顺序
            3.浏览器默认样式优先级最低
    5.选择器
        1.作用：根据不同的方式匹配文档中的元素，设置样式
        2.分类：
            1.标签选择器(元素选择器)
                作用：根据指定的标签名匹配文档中所有该元素
                语法：
                    标签名{
                        属性:值;
                    }
            2.id选择器
                作用：根据元素的id属性值匹配文档元素，id具有唯一性，不能重复使用
                语法：
                    #id属性值{
                        样式 
                    }
                    <标签名 id="">
                id属性值的命名规范：
                    可以由数字、字母、下划线、-（连接符）组成，禁止以数字开头
                    命名尽量见名知意，多个单词组成时，采用小驼峰
            3.class类选择器
                作用：根据class属性值匹配元素，类名可以重复使用，实现样式的复用
                语法：
                    <div class="c1"></div>   
                    .c1{
                        样式 
                    }
                特殊用法：
                    1.标签选择器与类选择器结合使用，精确匹配同时满足标签名与类名的元素，要求必须标签名在前，类名在后
                    2.标签属性不能重复书写，class属性值可以取多个，应用不同类选择器的样式，属性值之间使用空格隔开
                        例：class="c1 c2"
            4.群组选择器
                作用：同时为多组选择器对应的元素设置样式
                语法： 
                    选择器1,选择器2,...选择器n{
                        样式 
                    }       
            5.后代选择器
                作用：依据元素嵌套结构，匹配所有后代元素
                语法： 
                    选择器1 选择器2{
                        样式
                    }
                    匹配选择器1对应元素内部所有满足选择器2的后代元素
            6.子代选择器
                作用：匹配直接子元素，不考虑间接子元素
                语法： 
                    选择器1>选择器2{
                        样式 
                    }
