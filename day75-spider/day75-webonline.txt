    2.Web服务
        1.nginx:高并发处理的好
        2.Apache:稳定
            LAMP : Linux 、Apache、MySQL、Python/php/per1
            LNMP : Linux 、Nginx、MySQL、Python/php/per1 
        3.uwsgi
            是web服务器与web框架之间一种简单而通用的接口

2.项目部署
    1.部署环境准备
        1.确保Django项目可以运行
        2.安装nginx
            1.安装：sudo apt-get install nginx
            2.启动：sudo /etc/init.d/nginx restart
            3.验证：浏览器，输入127.0.0.1
        3.安装uwsgi
            1.安装：sudo pip3/apt-get install uwsgi
            2.验证：uwsgi --http :9999 --chdir /项目地址/ --module 主包名.wsgi
    2.项目部署步骤  
        1.配置uwsgi
            1.在项目目录(manage.py所在路径)中新建uwsgi启动文件：blogsUwsgi.ini 
                1.指定和nginx通信的端口
                    socket = 127.0.0.1:9001
                2.指定项目路径
                    chdir = /home/tarena/project/fruitday
                3.wsgi.py路径
                    wsgi-file = fruitday/wsgi.py 
                4.进程数
                    processes = 4
                5.线程数
                    thread = 2
                6.uwsgi自身占用的端口
                    stats = 127.0.0.1:8080
        2.配置nginx
            1.sudo -i 
            2.cd /etc/nginx/sites-enabled/
            3.vi projectNginx.conf
            4.拷贝uwsgi_params文件到项目目录
            5.重启nginx服务
                sudo /etc/init.d/nginx restart
        
            server{
                1,2,3
            }
            1.监听项目端口，浏览器输入端口
                listen 80;
                server_name www.xulinblogs.cn;
                charset utf-8;
            2.收集静态文件的路径
                location /static{
                    alias /static文件夹的地址;
                }
            3.和uwsgi通信端口 和项目通信的文件
                location /{
                    include uwsgi_params;
                    uwsgi_pass 127.0.0.1:9001;
                }

        3.收集静态文件
            1.在settings.py 中添加路径(STATIC_ROOT)
                STATIC_ROOT = "静态文件绝对路径"
            2.收集静态文件 
                python manage.py collectstatic
        
        4.添加本地解析
            1.sudo -i 
            2.vi /etc/hosts
                127.0.0.1 www.tiger.com
            3.重启网络服务
                /etc/init.d/networking restart
        4.uwsgi启动项目
            1.切换到项目目录
                cd /home/tarena/project/fruitday 
            2.利用uwsgi启动项目
                uwsgi --ini fruitdayUwsgi.ini
