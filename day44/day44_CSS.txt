伪类选择器
    作用：针对元素不同状态设置不同状态
    分类：
        1.超链接伪类选择器
            针对超链接不同状态设置样式
            ：link  超链接访问前
            ：visited  超链接访问后
        2.动态伪类选择器
            ：hover  鼠标滑过或悬停在元素时的状态
            ：active  激活状态（鼠标点按不抬起）
            ：focus  焦点状态（输入框在编辑状态时就是获取焦点）
    使用：
        必须与基础选择器（标签 id class）结合使用
        语法：
            a:link{}
            div:hover{}
    超链接四种状态的设置
        书写顺序
            :link
            :visited
            :hover
            :active
            简称love/hate爱恨原则
    超链接在页面中的样式设置
        a{//统一设置}
        a:hover{//单独设置鼠标滑过或停留的状态}

3.选择器的优先级
    不同选择器同时为元素设置样式，优先级看选择器权重（权值），值越大选择器的优先级越高
    选择器                  权重
    标签                    1
    class/伪类              10
    id                      100
    行内样式                1000
    
    组合选择器最终的权重由各选择器权值相加得到（群组选择器除外）      

4.HTML 元素分类
    1.块元素
        特点：
            1.独占一行，不与其他元素共行
            2.可以手动设置宽高，默认宽高与父元素保持一致
        例：
            h1~h6, p, div, ul, ol, li, form, table
        注：
            表格table默认的尺寸是由内容决定的
    2.行内元素
        特点：
            1.可以共行显示
            2.不能手动设置宽高
        例：
            span, label, b, strong, i, s, u, sub, sup, a 
    3.行内块元素
        特点：
            1.可以共行显示
            2.可以手动设置宽高 
        例：    
            img, 表单控件, input

5.标签嵌套
    1.块元素中可以嵌套任意类型的元素
        特殊情况：
            段落中不能嵌套标题
            只能在标题中写段落
    2.行内元素中尽量只嵌套行内或行内块

6.尺寸和颜色的取值
    1.尺寸单位
        1.px 浏览器的默认单位 像素
        2.%  百分比
        3.em 参照页面默认字体大小  1em=16px
        4.rem  参照页面根元素的字体大小
    2.颜色取值
        1.英文单词
        2.rgb(r, g, b) 采用三原色组合表示颜色，每一个三原色的取值范围 0~255
            特殊值：
                rgb(0, 0, 0)        黑色
                rgb(255, 255, 255)  白色
        3.rgba(r, g, b, a)  三原色设置颜色，alpha设置透明度，取值0（透明）~1（不透明）
        4.颜色的十六进制表示
            语法： 
                以#开头，每两位为一组，代表一种三原色
                每位取值范围 0~9 a~f 
                rgb(0, 0, 0)  --->  #000000  black
            特殊：
                #000
                #fff
                三位字符表示三原色
                浏览器解析时会自动对每一位进行重复，补全成长十六进制

7.内容与边框
    1.内容
        1.尺寸属性
            width/height：取像素值或百分比，设置内容尺寸
        2.内容溢出
            元素的内容部分超出了尺寸范围，称为溢出。默认情况下，溢出部分仍然可见
            属性：overflow
            取值：
                1.visible 默认值，溢出部分仍然可见
                2.hidden 溢出部分设置隐藏不可见
                3.scroll 强制在水平和垂直两个方向添加滚动条，不管当前方向是否发生内容溢出
                4.auto 自动在发生内容溢出的方向添加可用的滚动条
    2.边框
        1.边框实现
            属性：border
            作用：为元素设置上右下左四条边框
            取值：border-width border-style border-color;
                1.border-width: 边框宽度，默认3px，可以手动设置，取像素值
                2.border-style: 边框样式，必填项
                    可取：
                        solid   实线边框
                        dashed  虚线边框
                        dotted  点线边框
                        double  双线边框
                3.border-color: 边框颜色，取颜色值，默认为黑色,transparent为透明
        2.单边框设置
            属性：
                border-top
                border-right
                border-botton
                border-left
            取值：border-width border-style border-color;
        3.网页三角标的制作
            1.块元素设置宽高为0
            2.四个方向添加边框，宽度保持一致，拼接处三角形
            3.根据需要调整边框颜色，保留一条边框的可见色，其余为透明
            注意：三角标靠边框互相拼接而成，不能随意省略某一条边框
    3.轮廓线
        属性：outline
        取值：color style width
        特殊用法： 
            取消轮廓线 outline:none;
        轮廓线与边框线的区别：
            边框在文档中实际占位，会影响元素的最终尺寸
            轮廓线只具备显示效果，并不占位
    4.圆角边框
        1.作用： 
            设置元素四个角的显示形状或改变元素形状
        2.属性： 
            border-radius
        3.取值： 
            取像素值或百分比，表示圆角半径，以给出的半径在元素的四个角做圆或椭圆，取一段弧形成圆角效果
        4.取值情况
            1.像素值
                border-radius : 20px;
                    一个值表示上右下左四个角都以20px为半径取弧
                border-radius : 10px 20px 30px 40px;
                    四个值分别对应上右下左四个角的圆角半径
                    注： 浏览器或元素中，默认以左上角为原点
                border-radius : 20px 50px;
                    两个值分别代表 上下 左右，对角一致
                border-radius : 20px 30px 40px;
                    三个值的情况下，左右保持一致
            2.百分比
                参照元素自身尺寸计算圆角半径
                圆角半径 = width/height * %
                原理：根据宽高尺寸计算圆角半径，分别在水平和垂直方向取半径，确定弧
                取值：
                    border-radius : 20%;
                    border-radius : 20% 30%;
                    border-radius : 20% 30% 40%;
                    border-radius : 20% 30% 40% 50%; 
            3.取值范围
                1.统一设置四个角圆角效果，最大不能超过50%，即使代码书写超过50%，展示效果仍为50%
                2.如果统一设置50%，会改变元素形状，呈现正圆或椭圆（具体以元素的宽高尺寸为准）
                3.单个角设置圆角半径，取值范围[0%,100%)，100%等价于50%
    5.盒阴影
        属性：box-shadow
        取值： 
            offsetX: 阴影的水平偏移距离，取像素值
            offsetY: 阴影的垂直偏移距离，取像素值
            blur: 阴影的模糊程度，取像素值，值越大，越模糊
            spread: （选填）阴影的延伸距离，取像素值
            color: 阴影的颜色，默认为黑色，取颜色值
        坐标系：
            不管是浏览器窗口还是元素本身，都带有坐标系，默认都以左上角为(0, 0)点，向右向下分别表示X轴和Y轴的正方形
        阴影偏移：
            参考元素自身坐标系（元素左上角为原点），默认情况下阴影垂直显示在元素下方。偏移值，取正代表正方向偏移，负值代表向负方向偏移（向上向左）

8.盒模型
    1.一切元素皆为框
        组成（从外向内）：
            margin 外边距
            border 边框 
            padding 内边距
            width/height 内容框 
        作用：
            盒模型通过相关的CSS属性描述元素在文档中占据的最终尺寸，影响页面的排版和布局
        计算：
            大部分元素在文档中的实际尺寸由各个属性值相加计算 
    2.margin(外边框)
        作用：调整元素与元素之间的距离
        取值：像素值
            margin:20px;
                上右下左四个方向分别设置20px的外边距
            margin:20px 30px 40px 50px;
